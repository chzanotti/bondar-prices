# bondar-prices v5

Descarga TODOS los bonos USD de IOL automaticamente.
Funciona con cualquier ticker que cargues en BondAR.

## Setup
1. Fork este repo (boton arriba a la derecha)
2. Settings -> Pages -> Branch: main -> / (root) -> Save
3. Actions -> habilitar -> "Scrape IOL Prices" -> Run workflow
4. Esperar 5 minutos
5. URL: https://TU_USUARIO.github.io/bondar-prices/prices.json
6. En BondAR: boton "GitHub URL" -> pegar URL -> "Precios IOL"
